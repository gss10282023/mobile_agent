# Contributing

Thanks for your interest in Mobile Agent. Please keep contributions focused and
well-scoped so they are easy to review.

## How to contribute
1. Open an issue to discuss the change or bug.
2. Fork the repo and create a feature branch.
3. Keep changes minimal and update docs/tests where relevant.
4. Run tests if available (for example: `pytest`).
5. Submit a PR with a clear description, screenshots/logs if applicable, and
   device details for UI-related changes.

## Code style
- Prefer small, readable functions.
- Avoid large refactors unless discussed first.
- Add type hints where it improves clarity.

By participating, you agree to follow the Code of Conduct.
